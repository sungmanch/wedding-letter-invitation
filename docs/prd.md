# 모바일 청첩장 - Product Requirements Document

**Author:** BMad
**Date:** 2025-12-05
**Version:** 2.0

---

## Executive Summary

개인화된 모바일 청첩장 생성 서비스. 기존의 뻔한 템플릿 기반 청첩장 서비스와 달리, AI가 커플의 스토리와 선호도를 바탕으로 유니크한 청첩장 디자인을 생성해준다. 25개 이상의 섹션 타입과 11개의 인트로 스타일로 완전히 맞춤화된 청첩장을 만들 수 있으며, 게스트들은 축하 메시지를 통해 특별한 순간을 함께 나눌 수 있다.

**타겟 사용자:** 남들과 다른 특별한 청첩장을 원하는 예비 신부

**비즈니스 모델:** 청첩장 건당 결제 (9,900원)

### What Makes This Special

기존 모바일 청첩장 서비스들은 수십 개의 미리 만들어진 템플릿 중에서 선택하는 방식이다. 아무리 예쁜 템플릿이라도 "어디서 본 것 같은" 느낌을 지울 수 없다.

이 서비스는 다르다:
- **AI 디자인 생성**: 프롬프트 또는 사진 기반으로 5개의 유니크한 디자인 프리뷰 생성
- **11개 인트로 스타일**: Cinematic, Magazine, Vinyl, Passport 등 다양한 첫인상
- **25개 섹션 타입**: 영상, 타임라인, 인터뷰, D-DAY 등 풍부한 콘텐츠
- **2패널 실시간 편집**: 정보 입력과 동시에 실시간 프리뷰 확인
- **게스트 축하 메시지**: 청첩장을 받은 게스트가 축하 편지를 보낼 수 있음

## Project Classification

**Technical Type:** web_app
**Domain:** general
**Complexity:** medium

모바일 우선 웹 애플리케이션으로, 복잡한 규제나 도메인 특화 요구사항이 없는 일반 소비자 대상 서비스이다. 핵심 기술 챌린지는 AI 기반 디자인 생성, 확장 가능한 섹션 시스템, 실시간 편집 경험에 있다.

## Success Criteria

### MVP 성공 정의

**MVP 성공 = 20명 결제 완료**

| 지표 | 목표 | 기간 | 전환율 |
|------|------|------|--------|
| 회원가입 | 100명 | 1개월 | - |
| 결제 완료 | 20명 | 1개월 | 20% |

### 핵심 전환 퍼널

```
방문 → 회원가입 → 이미지 업로드/테마 선택 → AI 프리뷰 생성 → 디자인 선택
→ 2패널 편집 → 미리보기 확인 → 결제 완료 → 공유
```

**핵심 전환 지표:** 결제 완료

### 사용자 획득 전략

**1. SEO (Primary)**
- 타겟 키워드: "모바일 청첩장", "청첩장 만들기", "개인화 청첩장", "AI 청첩장"
- 콘텐츠: 청첩장 작성 가이드, 트렌드 소개

**2. 입소문 / 바이럴 루프 (Secondary)**
- 청첩장을 받은 게스트가 서비스 인지
- 본인 결혼 시 자연스럽게 재방문
- 청첩장 하단에 "Made with [서비스명]" 워터마크/링크

## Product Scope

### MVP - Minimum Viable Product

AI 기반 개인화 청첩장의 핵심 가치를 검증하기 위한 기능 세트

**1. 청첩장 기본 정보 입력**
- 신랑/신부 이름
- 신랑/신부 혼주 이름
- 결혼 일시
- 결혼 장소 (오시는 길 지도 포함)
- 연락처
- 축의금 계좌번호

**2. AI 디자인 생성**
- 이미지 업로드 기반 AI 분석 (선택)
- 프롬프트 기반 디자인 생성
- 5개 디자인 프리뷰 동시 생성
- 선택된 프리뷰 기반 상세 레이아웃 생성

**3. 인트로 스타일 (11개)**
- Keynote (Apple 스타일)
- Cinematic (영화적)
- Exhibition (갤러리 전시)
- Magazine (매거진 커버)
- Vinyl (LP 레코드)
- Chat (메신저 스타일)
- Glassmorphism (글래스모피즘)
- Passport (여권 스타일)
- Pixel (8비트 레트로)
- Typography (키네틱 타이포)
- None (인트로 없음)

**4. 확장 섹션 시스템 (25개 타입)**
- 기본: Hero, Greeting, Calendar, Gallery, Location, Parents, Contact, Account, Message, RSVP, Closing
- 확장: Video, Interview, Timeline, D-DAY, Profile, Quote, Transport, Notice, Announcement, Flower-Gift, Together-Time, Guest-Snap, Ending

**5. 2패널 편집 시스템**
- 좌측: 정보 입력 폼
- 우측: 실시간 프리뷰
- 섹션 순서 변경
- 섹션별 설정 커스터마이징

**6. 사진 업로드**
- 최대 10장 업로드
- 드래그앤드롭 순서 변경
- 디자인에 맞는 갤러리 레이아웃 자동 적용

**7. 게스트 축하 메시지**
- 닉네임 필드 (필수)
- 300자 제한
- Host만 열람 가능 (비공개)

**8. 미리보기 및 결제**
- 결제 전 전체 미리보기 확인
- 결제 후에도 내용 수정 가능
- 건당 9,900원 결제

**9. 청첩장 공유**
- 카카오톡 공유
- 링크 복사

### Growth Features (Post-MVP)

**확장 기능**
- 영상 청첩장 (전체 페이지 영상)
- RSVP 참석 확인 기능
- 템플릿 마켓플레이스
- B2B 웨딩 업체 제휴

## User Experience Principles

### Visual Identity

- **톤:** 로맨틱하고 따뜻한 웨딩 테마
- **Primary Color:** Deep Rose (#D4768A) - CTA 버튼
- **Accent Colors:** Blush Pink (#FFB6C1), Soft Gold (#D4AF37)
- **스타일:** 부드러운 곡선, 우아한 타이포그래피
- **Typography:** Pretendard (한글 최적화, 모던)

### Theme Categories (6개)

| 카테고리 | 설명 | 대표 키워드 |
|----------|------|------------|
| Cinematic | 영화적 | 시네마, 영화, 드라마틱 |
| Playful | 유쾌한 | 발랄한, 재미있는, 픽셀 |
| Artistic | 예술적 | 아트, 갤러리, 전시 |
| Classic | 클래식 | 전통, 고급, 우아한 |
| Modern | 모던 | 미니멀, 심플, 깔끔한 |
| Retro | 레트로 | 빈티지, LP, 복고풍 |

### Interaction Principles

- **모바일 First:** 터치 친화적 UI, 큰 버튼, 스와이프 제스처
- **빠른 AI 생성:** 5초 이내 프리뷰 생성, 로딩 UX 최적화
- **실시간 피드백:** 2패널 편집으로 즉각적 프리뷰
- **풍부한 애니메이션:** 20+ 애니메이션 타입 지원

### Key User Flows

**Host Flow (청첩장 생성):**
```
랜딩 → 회원가입 → 이미지 업로드(선택) → 테마 선택/AI 생성
→ 디자인 선택 → 2패널 편집 (정보 입력 + 실시간 프리뷰)
→ 미리보기 확인 → 결제 (9,900원) → 청첩장 공유
```

**Guest Flow (청첩장 열람):**
```
링크 클릭 → 인트로 화면 → 청첩장 열람 → 축하 메시지 작성 (선택) → 제출 완료
```

**Host Flow (축하 메시지 열람):**
```
로그인 → 내 청첩장 → 축하 메시지 목록 열람
```

## Functional Requirements

### 사용자 계정 및 인증

- **FR1:** 사용자는 이메일로 회원가입할 수 있다
- **FR2:** 사용자는 소셜 로그인(카카오)으로 회원가입할 수 있다
- **FR3:** 사용자는 로그인하여 세션을 유지할 수 있다
- **FR4:** 사용자는 비밀번호를 재설정할 수 있다

### 청첩장 기본 정보 입력

- **FR5:** Host는 새 청첩장을 생성할 수 있다
- **FR6:** Host는 신랑/신부 이름을 입력할 수 있다
- **FR7:** Host는 신랑/신부 혼주 이름을 입력할 수 있다
- **FR8:** Host는 결혼 일시를 입력할 수 있다
- **FR9:** Host는 결혼 장소 정보를 입력할 수 있다 (장소명, 주소, 지도)
- **FR10:** Host는 연락처를 입력할 수 있다 (신랑측, 신부측)
- **FR11:** Host는 축의금 계좌번호를 입력할 수 있다 (신랑측, 신부측)

### AI 디자인 생성

- **FR12:** Host는 원하는 청첩장 스타일을 텍스트로 설명할 수 있다
- **FR13:** 시스템은 텍스트 프롬프트를 기반으로 5개의 디자인 프리뷰를 생성한다
- **FR14:** Host는 5개 프리뷰 중 하나를 선택할 수 있다
- **FR15:** Host는 추가 결제 후 새로운 5개 시안을 재생성할 수 있다
- **FR36:** Host는 웨딩 사진을 업로드하여 AI가 분석하게 할 수 있다
- **FR37:** 시스템은 업로드된 이미지를 분석하여 무드/컬러/스타일을 추출한다

### 인트로 스타일

- **FR38:** Host는 11가지 인트로 스타일 중 선택할 수 있다
- **FR39:** Host는 인트로를 비활성화할 수 있다 (None 선택)
- **FR40:** 각 인트로는 스킵 기능을 제공한다

### 섹션 시스템

- **FR41:** Host는 25개 섹션 타입 중 원하는 섹션을 추가할 수 있다
- **FR42:** Host는 섹션 순서를 드래그앤드롭으로 변경할 수 있다
- **FR43:** Host는 각 섹션의 설정을 커스터마이징할 수 있다
- **FR44:** Host는 섹션을 숨기거나 삭제할 수 있다

### 확장 섹션

- **FR45:** Host는 Video 섹션에 YouTube/Vimeo 영상을 추가할 수 있다
- **FR46:** Host는 Timeline 섹션으로 커플 스토리를 보여줄 수 있다
- **FR47:** Host는 Interview 섹션으로 웨딩 Q&A를 추가할 수 있다
- **FR48:** D-DAY 섹션은 결혼일까지 실시간 카운트다운을 표시한다

### 2패널 편집 시스템

- **FR49:** 편집 화면은 좌측 폼 / 우측 프리뷰로 구성된다
- **FR50:** 폼 입력 시 우측 프리뷰가 실시간으로 업데이트된다
- **FR51:** Host는 프리뷰에서 섹션을 클릭하여 해당 폼으로 이동할 수 있다

### 사진 업로드

- **FR16:** Host는 최대 10장의 사진을 업로드할 수 있다
- **FR17:** 시스템은 선택된 디자인에 맞게 사진 레이아웃을 자동 적용한다
- **FR18:** Host는 업로드한 사진을 삭제하거나 교체할 수 있다
- **FR52:** Host는 드래그앤드롭으로 사진 순서를 변경할 수 있다

### 미리보기 및 결제

- **FR19:** Host는 결제 전 청첩장 전체 미리보기를 확인할 수 있다
- **FR20:** Host는 청첩장 결제(9,900원)를 완료할 수 있다
- **FR21:** Host는 결제 후에도 청첩장 내용을 수정할 수 있다

### 청첩장 공유

- **FR22:** Host는 카카오톡으로 청첩장을 공유할 수 있다
- **FR23:** Host는 청첩장 링크를 복사할 수 있다
- **FR24:** 청첩장에는 서비스 워터마크/링크가 표시된다 (바이럴)

### 게스트 청첩장 열람

- **FR25:** Guest는 로그인 없이 청첩장을 열람할 수 있다
- **FR26:** Guest는 청첩장에서 결혼 정보를 확인할 수 있다 (일시, 장소, 지도)
- **FR27:** Guest는 축의금 계좌번호를 복사할 수 있다
- **FR53:** Guest는 인트로 화면을 보고 스킵할 수 있다

### 게스트 축하 메시지

- **FR28:** Guest는 축하 메시지를 작성할 수 있다 (선택)
- **FR29:** Guest는 닉네임을 입력해야 한다 (필수)
- **FR30:** 축하 메시지는 300자로 제한된다
- **FR31:** Guest는 메시지 제출 후 수정할 수 없다

### Host 축하 메시지 관리

- **FR32:** Host는 받은 축하 메시지 목록을 열람할 수 있다
- **FR33:** 축하 메시지는 Host만 볼 수 있다 (비공개)
- **FR54:** Host는 메시지 읽음/안읽음 상태를 확인할 수 있다

### 청첩장 관리

- **FR34:** Host는 자신이 만든 청첩장 목록을 조회할 수 있다
- **FR35:** Host는 청첩장을 삭제할 수 있다

## Non-Functional Requirements

### Performance

| 항목 | 기준 |
|------|------|
| AI 프리뷰 생성 | 5초 이내 |
| AI 레이아웃 생성 | 10초 이내 |
| 페이지 로딩 | 모바일에서 3초 이내 |
| 이미지 업로드 | 10장 동시 업로드 지원 |
| LCP | 2.5초 이내 |

### Security

| 항목 | 설명 |
|------|------|
| 인증 | 이메일/카카오 OAuth 기반 인증 |
| 청첩장 URL | 추측 불가능한 UUID 사용 |
| 축하 메시지 | Host 외 접근 차단 (RLS) |
| 결제 | Polar.sh 연동 (민감 정보 미저장) |
| 개인정보 | 최소 수집 원칙 (이름, 연락처, 계좌) |

### SEO & Open Graph

| 항목 | 설명 |
|------|------|
| 서비스 페이지 | SEO 최적화 (메타 태그, 시맨틱 HTML) |
| 청첩장 페이지 | `noindex` 적용 (검색 노출 방지) |
| Open Graph | 카카오톡 공유 시 동적 미리보기 (신랑♥신부) |
| 타겟 키워드 | "모바일 청첩장", "청첩장 만들기", "AI 청첩장" |

### Reliability

| 항목 | 기준 |
|------|------|
| 가용성 | 99% 이상 |
| 데이터 백업 | DB 자동 백업 |
| 에러 처리 | 사용자 친화적 에러 메시지 |
| AI 폴백 | AI 실패 시 기본 프리뷰 5개 제공 |
